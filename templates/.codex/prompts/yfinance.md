# Codex カスタムプロンプト: yfinance で株価情報を取得

次の手順で米国株の最新情報を取得・要約してください。

1. Python で `yfinance` をインポートし、`Ticker("$1")` を使ってデータを取得する。`$1` にはコマンド引数で受け取ったティッカーシンボルを挿入する。
2. `info` や `history(period="5d")` を利用し、現在値・直近高値/安値・一日の変動率など主要指標を抽出する。
3. 結果を日本語で箇条書きし、投資判断やリスクに触れる場合は注意書きを添える。
4. 必要に応じて追加のツール実行（例えば `plotly` や `matplotlib`）を提案するが、自動実行は行わない。

実行例: `/prompts:yfinance AAPL`

※ 市場データは遅延している可能性があるため、必ず「情報は参考値」と明記してください。
