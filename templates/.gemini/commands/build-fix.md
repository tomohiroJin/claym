# ビルドエラー修正コマンド

以下の手順でビルドエラーを解決してください：

## 1. エラーの収集
プロジェクトの技術スタックに応じたビルドコマンドを実行してエラーを収集します。
`@package.json` や `@pyproject.toml` 等で技術スタックを確認してください。

追加コンテキスト: `$1`

## 2. エラーの分類
- 型・コンパイルエラー（型不一致、未定義シンボル等）
- インポート・依存エラー（モジュール未解決）
- 設定エラー（ビルド設定・コンパイラ設定）
- 依存関係エラー（パッケージ不整合）

## 3. 原因の特定
- エラーメッセージの意味を説明
- 根本原因を特定（連鎖エラーに注意）
- `@` 参照で関連設定を確認

## 4. 修正の実行
- 連鎖エラーを優先（1つ直すと複数解決するもの）
- 最小限の変更で解決
- 修正前後のコードを明示

## 5. 修正後の確認
プロジェクトのビルド・型チェック・テストコマンドを実行して確認します。

結果を報告してください。
