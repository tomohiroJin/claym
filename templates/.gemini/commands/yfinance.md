# Gemini カスタムコマンド: yfinance で株価情報を取得

次の手順で米国株の最新情報を取得・要約してください。

1. 引数 `$1` がティッカーか企業名かを判定する。企業名と思われる場合は `web_search` もしくは `fetch` ツールで「$1 ticker」などと検索し、正式なティッカーを特定する。候補が複数あるときは信頼できる情報源を引用し、最も妥当なものを選ぶ。
2. Python で `yfinance` をインポートし、決定したティッカーに対して `Ticker("<決定したティッカー>")` を使ってデータを取得する。引数がすでにティッカーの場合はそのまま利用する。
3. `info` や `history(period="5d")` を利用し、現在値・直近高値/安値・一日の変動率など主要指標を抽出する。
4. 結果を日本語で箇条書きし、使用したソース（検索結果の URL や公式サイトなど）も併記する。投資判断やリスクに触れる場合は注意書きを添える。
5. 必要に応じて追加のツール実行（例えば `plotly` や `matplotlib`）を提案するが、自動実行は行わない。

実行例: `/prompts:yfinance AAPL`

※ 市場データは遅延している可能性があるため、必ず「情報は参考値」と明記してください。
